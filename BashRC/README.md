Bashrc
======
Add these to your home folder (~/). The terminal will be colored, only the first directory will be displayed next to ~$ and
there's a ton of aliases in .bash_aliases.

* [See this Stackoverflow post for more info.](https://superuser.com/questions/49289/what-is-the-bashrc-file)
